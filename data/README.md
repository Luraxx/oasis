# Data Directory

This folder holds the input data for the pipeline. It is **not tracked by git** (too large).
Only the empty directory structure and this README are committed.

## Probability Maps

Each `.tif` is a single-band GeoTIFF with values 0–1000 (= probability × 1000).
- Resolution: 10m × 10m
- Size: ~1000×1000 pixels per tile
- CRS: UTM (varies per tile)

These are generated by the model training pipelines in `pipeline/02_models/`.

## Sentinel-2 Data

Monthly median composites, 12 bands (B01–B12), used only for year estimation (NBR drop analysis).
Download from the challenge data: https://github.com/osapiens/makeathon-challenge-2026

## Running the Pipeline

Once data is in place, run from `pipeline/`:

```bash
cd pipeline
python run_pipeline.py          # Reproduces Try 15 (53.46% IoU)
python run_pipeline.py --dry-run  # Stats only, no output
```

All paths are configured in `pipeline/config.py` relative to the repo root.
